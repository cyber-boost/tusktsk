# Peanuts Commands

Peanut configuration for TuskLang Ruby SDK.

## Available Commands

- [tsk peanuts compile](./compile.md) - Compile Peanut configuration
- [tsk peanuts auto-compile](./auto-compile.md) - Auto-compile Peanut files
- [tsk peanuts load](./load.md) - Load Peanut configuration
