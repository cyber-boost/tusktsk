# Binary Commands

Binary compilation for TuskLang Ruby SDK.

## Available Commands

- [tsk binary compile](./compile.md) - Compile to binary format
- [tsk binary execute](./execute.md) - Execute binary file
- [tsk binary benchmark](./benchmark.md) - Benchmark binary performance
- [tsk binary optimize](./optimize.md) - Optimize binary size and speed
