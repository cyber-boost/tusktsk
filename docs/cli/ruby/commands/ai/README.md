# Ai Commands

AI integrations for TuskLang Ruby SDK.

## Available Commands

- [tsk ai claude](./claude.md) - Interact with Claude AI
- [tsk ai chatgpt](./chatgpt.md) - Interact with ChatGPT
- [tsk ai custom](./custom.md) - Use custom AI provider
- [tsk ai config](./config.md) - Configure AI settings
- [tsk ai setup](./setup.md) - Setup AI API keys
- [tsk ai test](./test.md) - Test AI connections
- [tsk ai complete](./complete.md) - Complete code with AI
- [tsk ai analyze](./analyze.md) - Analyze code with AI
- [tsk ai optimize](./optimize.md) - Optimize code with AI
- [tsk ai security](./security.md) - Security analysis with AI
