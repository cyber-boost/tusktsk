# Examples: TuskLang JavaScript CLI

## Basic Usage
- Parsing a config: `tsk parse peanu.peanuts`
- Validating: `tsk validate peanu.peanuts`
- Compiling: `tsk config compile peanu.peanuts`

## Workflows
- Development: `tsk serve 3000`
- Testing: `tsk test all`
- Database: `tsk db status`
