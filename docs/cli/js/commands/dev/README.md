# Development Commands

Development tools for TuskLang JavaScript CLI.

## Available Commands

| Command | Description |
|---------|-------------|
| [tsk serve](./serve.md) | Start development server |
| [tsk compile](./compile.md) | Compile .tsk file |
| [tsk optimize](./optimize.md) | Optimize .tsk file for production |

## Common Use Cases
- Local development server
- Compiling TSK files
- Optimizing configuration for production

## JavaScript Notes
- Works with Node.js and Express.js
- Supports hot reload with file watching
