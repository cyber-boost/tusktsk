# Testing Commands

Testing tools for TuskLang JavaScript CLI.

## Available Commands

| Command | Description |
|---------|-------------|
| [tsk test all](./all.md) | Run all tests |
| [tsk test parser](./parser.md) | Test parser only |
| [tsk test fujsen](./fujsen.md) | Test FUJSEN only |
| [tsk test sdk](./sdk.md) | Test SDK only |
| [tsk test performance](./performance.md) | Performance tests |

## Common Use Cases
- Run all tests before deployment
- Test parser or SDK in isolation
- Benchmark performance

## JavaScript Notes
- Integrates with Mocha, Jest, or custom test runners
