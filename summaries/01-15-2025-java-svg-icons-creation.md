## Design Specifications

### Color Palette
- **Primary Orange:** #F89820 (Java brand)
- **Dark Orange:** #ED8B00 (Java brand)
- **Blue:** #007396 (Java brand)
- **Accent:** #4ECDC4 (for testing/success)
- **Error:** #FF6B6B (for exceptions)
- **Background:** #2A2A3E (dark theme)

### Technical Specs
- **ViewBox:** 64x64 pixels for all icons
- **Stroke Width:** Consistent 1.5-2px
- **Font:** Courier New for code-like appearance
- **Accessibility:** Title and desc tags included
- **Scalability:** Vector-based for any size

### Visual Consistency
- Consistent corner radius (3-4px)
- Uniform stroke widths
- Java brand color usage throughout
- Minimalist, clean design approach
- Dark mode friendly
